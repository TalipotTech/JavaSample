2.1

Conditional Statements

Make decisions in your code based on conditions

üîÄ What are Conditional Statements?

Conditional statements allow your program to make decisions and execute different code blocks based on whether certain conditions are true or false. Think of them as the "if-then" logic in everyday life.

üö¶

if Statement

Execute code only if a condition is true. The most basic form of decision-making.

üîÑ

if-else Statement

Choose between two paths: one if true, another if false using else.

üå≥

else if Ladder

Check multiple conditions in sequence using else if chains.

üéõÔ∏è

switch Statement

Select one of many code blocks based on a variable's value using switch.

üö¶ The if Statement

The simplest conditional statement. It executes a block of code only if the specified condition evaluates to true.

üìê Syntax

if (condition) { // Code to execute if condition is true }
IfStatement.java
Java 
public class IfStatement {

    public static void main(String[] args) {
        int age = 20;
        int temperature = 35;
        boolean hasTicket = true;
        
        // Simple if statement
        if (age >= 18) {
            System.out.println("You are an adult.");
        }
        
        // if with comparison operator
        if (temperature > 30) {
            System.out.println("It's a hot day! üåû");
            System.out.println("Stay hydrated!");
        }
        
        // if with boolean variable
        if (hasTicket) {
            System.out.println("Welcome to the show! üé≠");
        }
        
        // if with logical operators
        if (age >= 18 && hasTicket) {
            System.out.println("You can enter the venue.");
        }
        
        // Nested if statements
        if (age >= 18) {
            if (hasTicket) {
                System.out.println("Enjoy the concert!");
            }
        }
    }
}
üì§ Output
You are an adult. It's a hot day! üåû Stay hydrated! Welcome to the show! üé≠ You can enter the venue. Enjoy the concert!

üîÑ The if-else Statement

When you need to choose between two alternatives: execute one block if the condition is true, another block if it's false.

üìê Syntax

if (condition) { // Code if condition is TRUE } else { // Code if condition is FALSE }

            System.out.println(number + " is even");

        }
        
        // Pass or fail check

}

üí° Ternary Operator

The ternary operator condition ? valueIfTrue : valueIfFalse is a compact way to write simple if-else statements. Use it for simple assignments, but avoid it for complex logic as it reduces readability.

üå≥ The else if Ladder

When you have multiple conditions to check in sequence. The first condition that evaluates to true will execute its block, and all remaining conditions are skipped.

if (condition1) { // Code if condition1 is true } else if (condition2) { // Code if condition2 is true } else if (condition3) { // Code if condition3 is true } else { // Code if ALL conditions are false }

üéõÔ∏è The switch Statement

The switch statement is perfect when you need to compare a single variable against multiple possible values. It's cleaner and often more efficient than long else-if chains.

üìê Traditional Syntax

switch (expression) { case value1: // Code for value1 break; case value2: // Code for value2 break; default: // Code if no case matches }
SwitchExample.java
Java 
public class SwitchExample {

    public static void main(String[] args) {
        int dayNumber = 3;
        String dayName;
        
        // Traditional switch statement
        switch (dayNumber) {
            case 1:
                dayName = "Monday";
                break;
            case 2:
                dayName = "Tuesday";
                break;
            case 3:
                dayName = "Wednesday";
                break;
            case 4:
                dayName = "Thursday";
                break;
            case 5:
                dayName = "Friday";
                break;
            case 6:
                dayName = "Saturday";
                break;
            case 7:
                dayName = "Sunday";
                break;
            default:
                dayName = "Invalid day";
        }
        System.out.println("Day " + dayNumber + " is " + dayName);
        
        // Switch with String (Java 7+)
        String month = "March";
        int days;
        
        switch (month) {
            case "January":
            case "March":
            case "May":
            case "July":
            case "August":
            case "October":
            case "December":
                days = 31;
                break;
            case "April":
            case "June":
            case "September":
            case "November":
                days = 30;
                break;
            case "February":
                days = 28; // Not accounting for leap year
                break;
            default:
                days = 0;
        }
        System.out.println(month + " has " + days + " days");
    }
}

Switch Expressions (Java 14+)

Modern Java introduced switch expressions with arrow syntax - cleaner, no fall-through, and can return values directly.

ModernSwitch.java
Java 14+ 
public class ModernSwitch {

    public static void main(String[] args) {
        int dayNumber = 5;
        
        // Switch expression with arrow syntax
        String dayType = switch (dayNumber) {
            case 1, 2, 3, 4, 5 -> "Weekday";
            case 6, 7 -> "Weekend";
            default -> "Invalid";
        };
        System.out.println("Day " + dayNumber + " is a " + dayType);
        
        // Switch expression with blocks
        String season = "Summer";
        String activity = switch (season) {
            case "Spring" -> {
                System.out.println("üå∏ Flowers are blooming!");
                yield "Go hiking";
            }
            case "Summer" -> {
                System.out.println("‚òÄÔ∏è It's hot outside!");
                yield "Go swimming";
            }
            case "Fall", "Autumn" -> {
                System.out.println("üçÇ Leaves are falling!");
                yield "Go apple picking";
            }
            case "Winter" -> {
                System.out.println("‚ùÑÔ∏è It's cold outside!");
                yield "Go skiing";
            }
            default -> "Stay home";
        };
        System.out.println("Suggested activity: " + activity);
        
        // Pattern matching in switch (Java 21+)
        Object value = 42;
        String result = switch (value) {
            case Integer i when i > 0 -> "Positive integer: " + i;
            case Integer i -> "Non-positive integer: " + i;
            case String s -> "String: " + s;
            case null -> "Null value";
            default -> "Unknown type";
        };
        System.out.println(result);
    }
}
üì§ Output
Day 5 is a Weekday ‚òÄÔ∏è It's hot outside! Suggested activity: Go swimming Positive integer: 42
‚ö†Ô∏è Don't Forget break!

In traditional switch statements, forgetting break causes "fall-through" where execution continues to the next case. The modern arrow syntax (->) doesn't have this problem.

2.2

Loops

Execute code repeatedly until a condition is met

üîÑ What are Loops?

Loops allow you to execute a block of code multiple times. Instead of writing repetitive code, loops make your programs efficient and maintainable.

Loop Type Best For Condition Check
for Known number of iterations Before each iteration
while Unknown iterations, condition-based Before each iteration
do-while Must execute at least once After each iteration
for-each Iterating through collections/arrays Automatic

üî¢ The for Loop

The most commonly used loop when you know exactly how many times you want to iterate. It has three parts: initialization, condition, and update.

üìê Syntax

for (initialization; condition; update) { // Code to repeat } // Example: for (int i = 0; i < 5; i++) { // Runs 5 times (i = 0, 1, 2, 3, 4) }
ForLoopExamples.java
Java 
public class ForLoopExamples {

    public static void main(String[] args) {

        System.out.println("=== Counting 1 to 5 ===");
        for (int i = 1; i <= 5; i++) {
            System.out.println("Count: " + i);
        }
        
        // Counting down
        System.out.println("\n=== Countdown ===");
        for (int i = 5; i >= 1; i--) {
            System.out.println(i + "...");
        }
        System.out.println("üöÄ Liftoff!");
        
        // Stepping by 2
        System.out.println("\n=== Even numbers 2-10 ===");
        for (int i = 2; i <= 10; i += 2) {
            System.out.print(i + " ");
        }
        System.out.println();
        
        // Multiplication table
        System.out.println("\n=== 5x Table ===");
        for (int i = 1; i <= 10; i++) {
            System.out.println("5 x " + i + " = " + (5 * i));
        }
        
        // Sum of numbers
        int sum = 0;
        for (int i = 1; i <= 100; i++) {
            sum += i;
        }
        System.out.println("\nSum of 1-100: " + sum);
        
        // Nested for loops - pattern printing
        System.out.println("\n=== Star Pattern ===");
        for (int i = 1; i <= 5; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print("‚≠ê");
            }
            System.out.println();
        }
    }
}

üîÅ The while Loop

Executes a block of code as long as the condition is true. Best when you don't know in advance how many iterations you need.

üìê Syntax

while (condition) { // Code to repeat while condition is true // Make sure to update variables to eventually exit! }
WhileLoopExamples.java
Java 
public class WhileLoopExamples {

    public static void main(String[] args) {
        // Basic while loop
        System.out.println("=== Basic While Loop ===");
        int count = 1;
        while (count <= 5) {
            System.out.println("Count: " + count);
            count++;  // Don't forget to increment!
        }
        
        // Finding digits in a number
        System.out.println("\n=== Count Digits ===");
        int number = 12345;
        int original = number;
        int digitCount = 0;
        
        while (number > 0) {
            number = number / 10;  // Remove last digit
            digitCount++;
        }
        System.out.println(original + " has " + digitCount + " digits");
        
        // Sum digits of a number
        System.out.println("\n=== Sum of Digits ===");
        number = 12345;
        int digitSum = 0;
        
        while (number > 0) {
            int digit = number % 10;  // Get last digit
            digitSum += digit;
            number = number / 10;     // Remove last digit
        }
        System.out.println("Sum of digits in 12345: " + digitSum);
        
        // Finding GCD using Euclidean algorithm
        System.out.println("\n=== GCD Calculator ===");
        int a = 48, b = 18;
        int origA = a, origB = b;
        
        while (b != 0) {
            int temp = b;
            b = a % b;
            a = temp;
        }
        System.out.println("GCD of " + origA + " and " + origB + " is " + a);
        
        // Reverse a number
        System.out.println("\n=== Reverse Number ===");
        number = 12345;
        int reversed = 0;
        
        while (number > 0) {
            int digit = number % 10;
            reversed = reversed * 10 + digit;
            number /= 10;
        }
        System.out.println("12345 reversed: " + reversed);
    }
}

‚ö†Ô∏è Infinite Loop Warning

Always ensure your while loop's condition will eventually become false. An infinite loop occurs when the condition never becomes false, freezing your program. Always update loop variables!

2.3

Branching Statements

Control the flow within loops using break and continue

‚èπÔ∏è The break Statement

The break statement immediately exits the current loop or switch statement. It's useful when you've found what you're looking for or need to exit early.

BreakExample.java
Java 
public class BreakExample {

    public static void main(String[] args) {
        // Search for a number in array
        System.out.println("=== Linear Search ===");
        int[] numbers = {5, 12, 8, 130, 44, 23, 99};
        int target = 130;
        int foundIndex = -1;
        
        for (int i = 0; i < numbers.length; i++) {
            System.out.println("Checking index " + i + ": " + numbers[i]);
            if (numbers[i] == target) {
                foundIndex = i;
                System.out.println("üéØ Found! Breaking out...");
                break;  // Exit loop immediately
            }
        }
        
        if (foundIndex != -1) {
            System.out.println("Found " + target + " at index " + foundIndex);
        } else {
            System.out.println("Not found");
        }
        
        // Break in while loop
        System.out.println("\n=== Find First Vowel ===");
        String word = "rhythm";
        char firstVowel = '\0';
        
        for (char c : word.toCharArray()) {
            if ("aeiouAEIOU".indexOf(c) != -1) {
                firstVowel = c;
                break;
            }
        }
        
        if (firstVowel != '\0') {
            System.out.println("First vowel in '" + word + "': " + firstVowel);
        } else {
            System.out.println("'" + word + "' has no vowels!");
        }
        
        // Breaking out of nested loops with labels
        System.out.println("\n=== Labeled Break ===");
        outer:
        for (int i = 1; i <= 3; i++) {
            for (int j = 1; j <= 3; j++) {
                System.out.println("i=" + i + ", j=" + j);
                if (i == 2 && j == 2) {
                    System.out.println("Breaking out of BOTH loops!");
                    break outer;  // Breaks out of outer loop
                }
            }
        }
        System.out.println("Done!");
    }
}

‚è≠Ô∏è The continue Statement

The continue statement skips the current iteration and moves to the next one. It's useful when you want to skip certain values without exiting the entire loop.

ContinueExample.java
Java 
public class ContinueExample {

    public static void main(String[] args) {
        // Print odd numbers only
        System.out.println("=== Odd Numbers 1-10 ===");
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                continue;  // Skip even numbers
            }
            System.out.print(i + " ");
        }
        System.out.println();
        
        // Skip negative numbers when calculating sum
        System.out.println("\n=== Sum Positive Numbers ===");
        int[] values = {5, -3, 10, -7, 15, 8, -2};
        int positiveSum = 0;
        
        for (int val : values) {
            if (val < 0) {
                System.out.println("Skipping negative: " + val);
                continue;
            }
            positiveSum += val;
            System.out.println("Added " + val + ", Sum: " + positiveSum);
        }
        System.out.println("Total of positives: " + positiveSum);
        
        // Skip specific characters
        System.out.println("\n=== Remove Vowels ===");
        String text = "Hello World";
        StringBuilder noVowels = new StringBuilder();
        
        for (char c : text.toCharArray()) {
            if ("aeiouAEIOU".indexOf(c) != -1) {
                continue;  // Skip vowels
            }
            noVowels.append(c);
        }
        System.out.println("Original: " + text);
        System.out.println("No vowels: " + noVowels);
        
        // FizzBuzz with continue
        System.out.println("\n=== FizzBuzz (1-15) ===");
        for (int i = 1; i <= 15; i++) {
            if (i % 3 == 0 && i % 5 == 0) {
                System.out.print("FizzBuzz ");
                continue;
            }
            if (i % 3 == 0) {
                System.out.print("Fizz ");
                continue;
            }
            if (i % 5 == 0) {
                System.out.print("Buzz ");
                continue;
            }
            System.out.print(i + " ");
        }
        System.out.println();
    }

public class GuessingGame {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int totalGames = 0;
        int totalWins = 0;
        boolean playAgain = true;
        
        // Welcome message
        System.out.println("‚ïî‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïê‚ïó");

                default -> { maxNumber = 100; maxAttempts = 7; }

                        System.out.println("(Way too low! ü•∂)");

                        System.out.println("(Cold ‚ùÑÔ∏è)");

                        System.out.println("(Warm üå°Ô∏è)");
                    } else if (diff > 5) {
                        System.out.println("(Hot! üî•)");
                    } else {
                        System.out.println("(Very close! üéØ)");
                    }
